Angular - 組件傳值

發表於 2021-04-29 更新於 2021-05-12 分類於 Angular Disqus：
父傳子、子傳父、傳來傳去傳到哪。

父傳子

  • 傳資料

父元件需要先宣告要傳到子元件的變數，

接著透過html來傳送給子元件，

使用 Input 宣告要接收父元件的變數

最後在子元件中使用 data 即可。

  • 傳事件

父元件需要先宣告要傳到子元件的事件，

接著透過html來傳送給子元件，

使用 Input 宣告要接收父元件的變數

最後在子元件中使用 this.functionn() 即可。

子傳父

需要先在子元件使用 Output 宣告要傳送到父元件的變數，及利用 EventEmitter 輸出屬性

觸發事件後，會將事件內容傳送出去

接著透過html來傳送給子元件

父元件接到後即可執行。
